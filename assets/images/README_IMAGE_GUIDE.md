# Image Folder Guide

The website now has an easy image folder: `assets/images/`.

To update an image, replace the file here with a new image using the **same filename**. The helper script `assets/js/image-overrides.js` will use these friendly files across the site.

Main folders:

- `brand/` — logo
- `banners/` — large hero / collection banners
- `products/jasmine/` — Jasmine product image
- `products/original/` — Original product images
- `products/turmeric/` — Turmeric product images
- `about/` — About Us and team images
- `home/` — Home page lifestyle/product images
