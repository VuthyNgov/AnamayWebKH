# Anamay static website

Use a local server to preview:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000/

Main pages:
- `/` or `/Home/`
- `/Collection/`
- `/About-us/`
- `/Product/Jasmine-Soap/`
- `/Product/Original-Soap/`
- `/Product/Tumeric-Soap/`

The top-level duplicate `.html` pages were removed so links do not accidentally open old unfixed versions.


## Updating images

Use `assets/images/` to update images. Replace the relevant file with the same filename. See `assets/images/README_IMAGE_GUIDE.md` for the full map.
